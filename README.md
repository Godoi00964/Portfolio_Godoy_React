# 🚀 Meu Portfólio Profissional

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido para demonstrar minhas habilidades como Desenvolvedor Front-End, apresentando meus projetos, formação e formas de contato de maneira moderna, responsiva e performática.

## 📱 Visualização do Projeto

O portfólio apresenta um design limpo e moderno, com foco na experiência do usuário (UX) e interface intuitiva (UI).

---

## ✨ Funcionalidades Principais

- **🏠 Home Dinâmica**: Introdução impactante com animações de entrada.
- **🙋 Sobre Mim**: Seção detalhada sobre minha trajetória e paixão por tecnologia.
- **📚 Disciplinas em Estudo**: Exibição em cards das tecnologias e conceitos que estou dominando.
- **📱 Responsividade Total**: Experiência otimizada para desktops, tablets e dispositivos móveis.
- **🍔 Menu Mobile**: Navegação intuitiva em telas menores com menu estilo hambúrguer.
- **✨ Scroll Reveal**: Animações suaves que surgem conforme o usuário navega pela página.
- **🏗️ Componentização**: Código modular e organizado, com destaque para o componente de Botão reutilizável.
- **📩 Formulário de Contato**: Interface pronta para interação com potenciais clientes ou recrutadores.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores práticas do desenvolvimento moderno:

- **React 19**: Biblioteca principal para construção da interface.
- **Vite**: Build tool extremamente rápida para o desenvolvimento.
- **CSS3 Moderno**: Utilização de Flexbox, CSS Grid e variáveis para um design flexível.
- **JavaScript (ES6+)**: Lógica e interatividade.
- **Bootstrap 5**: Auxílio na estrutura e componentes base.
- **Intersection Observer API**: Utilizada para as animações de scroll reveal.

## 🚀 Como Rodar o Projeto Localmente

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/portfolio.git
   ```

2. **Entre na pasta do projeto:**
   ```bash
   cd portfolio
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

5. **Acesse no navegador:**
   O projeto estará disponível em `http://localhost:5173`.
